Customize your WordPress dashboard and login page with your own branding, logo, and styling.

== Description ==

Custom WP Dashboard With Login allows you to fully customize the appearance of your WordPress admin dashboard and login page. Add your own branding, logos, and customize the look and feel to match your company or client's brand identity.

= Features =

* **Custom Dashboard Logo** - Replace the WordPress logo with your own
* **Custom Login Page** - Fully customizable login page design
* **Configurable Support Link** - Optionally add a support link in the admin bar
* **Custom Copyright Text** - Add your own copyright message in the dashboard footer
* **Clean Admin Interface** - Simplified admin menu for non-admin users
* **Easy to Configure** - Simple settings page to manage all options
* **No Coding Required** - All customization through the WordPress admin

= Usage =

1. Install and activate the plugin
2. Go to Dashboard Settings in the admin menu
3. Configure your logos, URLs, and branding
4. Save settings and your customizations will be applied

= Privacy =

This plugin does not collect, store, or transmit any user data. All settings are stored locally in your WordPress database.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/custom-wp-dashboard-with-login` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to Dashboard Settings to configure the plugin.
4. Add your custom logos, URLs, and branding information.
5. Save changes and enjoy your customized dashboard!

== Frequently Asked Questions ==

= Can I use my own logo? =

Yes! You can upload and use any logo you want for both the dashboard and login page.

= Does this plugin require coding knowledge? =

No, all customization is done through a simple settings page in the WordPress admin.

= Will this work with my theme? =

Yes, this plugin works independently of your theme and customizes the WordPress admin area only.

= Can I hide the support link? =

Yes, the support link is optional and can be disabled in the settings.

= Is the plugin translation ready? =

Yes, the plugin includes full internationalization support and can be translated to any language.
