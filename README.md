# Custom WP Dashboard With Login

A WordPress plugin that allows you to customize your WordPress dashboard and login page with your own branding, logo, and styling.

## Features

- **Custom Dashboard Logo** - Replace the WordPress logo with your own
- **Custom Login Page** - Fully customizable login page design
- **Configurable Support Link** - Optionally add a support link in the admin bar
- **Custom Copyright Text** - Add your own copyright message in the dashboard footer
- **Clean Admin Interface** - Simplified admin menu for non-admin users
- **Easy to Configure** - Simple settings page to manage all options
- **No Coding Required** - All customization through the WordPress admin

## Installation

1. Upload the plugin files to the `/wp-content/plugins/custom-wp-dashboard-with-login` directory, or install through WordPress
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to Dashboard Settings to configure the plugin
4. Add your custom logos, URLs, and branding information
5. Save changes and enjoy your customized dashboard!

## Requirements

- WordPress 6.0 or higher
- PHP 7.4 or higher

## Usage

After activation, go to **Dashboard Settings** in your WordPress admin menu. From there you can:

1. Set your dashboard logo URL
2. Set your login page logo URL
3. Configure your website URL
4. Enable/disable and configure the support link
5. Set custom copyright text
